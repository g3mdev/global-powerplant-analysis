# global-powerplant-analysis

This project develops a powerplant management system that processes global infrastructure energy data to model the environmental impact of different energy types.  Using Python (Pandas, Matplotlib, pure Python loops & vectorised NumPy operations), I applied OOP principles to build a queryable registry, benchmarked pure Python & vectorised NumPy performance and visualised differences in space/time complexity. It demonstrates my ability to clean, analyse and present data effectively.
